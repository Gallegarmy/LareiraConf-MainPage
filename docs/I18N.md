# Sistema de Internacionalización (i18n)

Este proyecto utiliza un sistema de internacionalización para soportar múltiples idiomas (Español y Gallego).

## Estructura

```
src/i18n/
├── es.json          # Traducciones en español
├── gl.json          # Traducciones en gallego
└── utils.ts         # Utilidades para gestionar traducciones
```

## Configuración

La configuración de i18n está en `astro.config.mjs`:

```javascript
i18n: {
  defaultLocale: "es",
  locales: ["es", "gl"],
  routing: {
    prefixDefaultLocale: false
  }
}
```

## Uso en componentes de Astro

```astro
---
import { getLangFromUrl, useTranslations } from '@/i18n/utils';

const lang = getLangFromUrl(Astro.url);
const t = useTranslations(lang);
---

<h1>{t('faq.title')}</h1>
```

## Uso en componentes de React

Para componentes React, pasa el idioma como prop desde el componente padre de Astro:

```tsx
import { type Locale } from '@/i18n/utils';

interface MyComponentProps {
  locale: Locale;
}

const MyComponent: React.FC<MyComponentProps> = ({ locale }) => {
  // Usa locale para obtener las traducciones necesarias
};
```

## Añadir nuevas traducciones

1. Añade la clave en `es.json`:
```json
{
  "newSection": {
    "title": "Título en español",
    "description": "Descripción en español"
  }
}
```

2. Añade la traducción en `gl.json`:
```json
{
  "newSection": {
    "title": "Título en galego",
    "description": "Descrición en galego"
  }
}
```

3. Usa en tu componente:
```astro
{t('newSection.title')}
{t('newSection.description')}
```

## Selector de idioma

El componente `LanguageSelector` está disponible para cambiar entre idiomas:

```astro
---
import LanguageSelector from '@components/LanguageSelector/LanguageSelector';
import { getLangFromUrl } from '@/i18n/utils';

const lang = getLangFromUrl(Astro.url);
---

<LanguageSelector currentLang={lang} client:load />
```

## Rutas

- **Español** (por defecto): `/` o `/es/`
- **Gallego**: `/gl/`

El sistema redirige automáticamente según el idioma seleccionado.
