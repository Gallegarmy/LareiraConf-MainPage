# Logos de Comunidades

Este directorio contiene los logos de las comunidades colaboradoras de Lareira Conf.

## Logos necesarios:

1. **corunha-wtf.png** - CoruñaWTF
2. **sysarmy-galicia.png** - SysarmyGalicia
3. **python-corunha.png** - Python Coruña
4. **gpul.png** - GPUL
5. **bricolabs.png** - Bricolabs
6. **datola.png** - Datola
7. **wordpress.png** - WordPress Coruña
8. **industriosa.png** - Industriosa
9. **corunha-jug.png** - CoruñaJUG
10. **aquelarre-dixital.png** - Aquelarre Dixital

## Especificaciones recomendadas:

- **Formato**: PNG con fondo transparente (preferible) o JPG/WEBP
- **Tamaño**: Min 400px de ancho, relación de aspecto 16:9 recomendada
- **Peso**: Optimizado para web (< 100KB por imagen)
- **Nombre**: Usar el formato especificado arriba (lowercase, guiones)

## Cómo añadir logos:

1. Guardar los archivos con los nombres especificados arriba
2. Colocarlos en este directorio (`public/communities/`)
3. Los logos se mostrarán automáticamente en la sección de comunidades
