# Página de Información para Patrocinadores

Esta página proporciona toda la información necesaria para los patrocinadores de LareiraConf 2026.

## 📍 URL

La página está disponible en: `/sponsors-info`

## 📝 Contenido

La página incluye información sobre:

### 🔥 La Previa (Viernes 20 de Marzo)
- Horario: 18:30 a 22:00
- Ubicación: Bellini Rooftop Restaurant & Bar
- Primera ronda por nuestra cuenta
- No habrá stands (excepto item especial)
- Actividades: Analítica, Videojuegos y Recruitment

### 🛠️ Día Principal (Sábado 21 de Marzo)
- Ubicación: Rectorado - UDC
- Hora de inicio: 10:00
- Acceso para montaje: desde las 8:00
- Incluye: Mesa, Sillas, Punto de electricidad
- Track único con charlas, entrevistas y show sorpresa

### 🎉 Post-evento (Sábado 21 de Marzo)
- Ubicación: MALTE Atochas
- Horario: a partir de las 21:00
- Incluido en entradas Full Pass
- Fiesta y cena para cerrar el evento

### 🎟️ Gestión de Entradas
- Formulario para solicitar entradas de patrocinador
- Enlace al dossier de patrocinio

## 🏗️ Estructura Técnica

### Archivos
- **Página**: `src/pages/sponsors-info.astro`
- **Componente React**: `src/components/SponsorsInfo/SponsorsInfo.tsx`
- **Estilos**: `src/components/SponsorsInfo/SponsorsInfo.scss`

### Tecnologías
- React + TypeScript
- SCSS para estilos
- Diseño responsivo
- Efectos hover y transiciones suaves

## 🎨 Diseño

La página sigue el sistema de diseño de LareiraConf 2026:
- Paleta de colores: #ff6b35 (naranja) y #f7931e (amarillo)
- Fondo oscuro con gradientes
- Glassmorphism en las cards
- Animaciones sutiles al hacer hover
- Completamente responsiva

## 🔄 Actualización

Para actualizar el contenido:
1. Edita el componente `src/components/SponsorsInfo/SponsorsInfo.tsx`
2. Modifica las secciones correspondientes
3. Los estilos se pueden ajustar en `SponsorsInfo.scss`
